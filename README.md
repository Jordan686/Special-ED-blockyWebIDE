# Visual web IDE for teaching HTML, CSS and JavaScript in a special education classroom

The project consisted of developing an IDE based on Google's Blockly library, customized to meet the requirements defined by a special education teacher. The goal was to enable its use in teaching web development to a class of students with learning disabilities. Additional features were also included to improve overall usability and convenience.


---


The web app was designed to be used with touch input, due to its ease of use and intuitive interaction.

## Hands-on application

With the IDE completed, I put it into practice by teaching two lessons to students under the supervision of a special education teacher. During these lessons, we covered basic HTML structures and styling, such as creating tables, navigation bars, and inputs.
