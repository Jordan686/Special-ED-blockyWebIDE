#  Visual web IDE for teaching HTML, CSS & JavaScript in a special education classroom

The project consisted of an IDE built using Google’s Blockly library as a foundation, but customized to meet the requirements defined by a special education teacher, so that it could be used to teach a class of students with learning disabilities about web development. It also included additional features to improve overall usability and convenience.


---

##  Informational Website

The website was designed to be used with touch input, due to the greater ease of use and intuitive interaction.

##  Hands-on action

With the IDE completed, it was applied at two lessons I gave to the students, under the supervision of a special education teacher. We went through basic HTML structures and styling, such as making tables, nav bars and inputs.
