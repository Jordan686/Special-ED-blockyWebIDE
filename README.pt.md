# IDE web visual para ensino de HTML, CSS e JavaScript em uma turma de educação especial

O projeto consistiu no desenvolvimento de uma IDE baseada na biblioteca Blockly, do Google, mas personalizada para atender aos critérios definidos por uma professora de educação especial. O objetivo foi possibilitar seu uso no ensino de desenvolvimento web para uma turma de alunos com deficiências de aprendizado. Também foram incluídas funcionalidades adicionais para melhorar a usabilidade e a conveniência geral.

Após o desenvolvimento da IDE, ministrei duas aulas utilizando a ferramenta, sob a supervisão da professora.

## Aplicação prática

Com a IDE finalizada, ela foi aplicada em duas aulas que ministrei para os alunos, sob a supervisão de uma professora de educação especial. Durante as aulas, foram abordadas estruturas básicas de HTML e estilização, como criação de tabelas, barras de navegação e inputs.
